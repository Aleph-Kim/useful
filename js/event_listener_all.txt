JS 이벤트 리스너 전체

1.	abort: 이미지 로딩 등이 중단될 때 발생하는 이벤트
2.	afterprint: 인쇄 후 문서가 나온 상태일 때 발생하는 이벤트
3.	animationend: CSS 애니메이션이 종료될 때 발생하는 이벤트
4.	animationiteration: CSS 애니메이션이 반복될 때 발생하는 이벤트
5.	animationstart: CSS 애니메이션이 시작될 때 발생하는 이벤트
6.	beforeprint: 인쇄 전 문서가 나올 상태일 때 발생하는 이벤트
7.	beforeunload: 페이지를 떠날 때 발생하는 이벤트 (브라우저가 종료되기 전)
8.	blur: 요소가 포커스를 잃을 때 발생하는 이벤트
9.	canplay: 미디어(오디오, 비디오 등)가 재생 가능할 때 발생하는 이벤트
10.	canplaythrough: 미디어가 전부 재생될 수 있을 때 발생하는 이벤트
11.	change: 입력 요소의 값이 변경될 때 발생하는 이벤트
12.	click: 요소를 클릭할 때 발생하는 이벤트
13.	contextmenu: 컨텍스트 메뉴를 표시하려 할 때 발생하는 이벤트
14.	copy: 복사하기 전에 발생하는 이벤트
15.	cut: 잘라내기 전에 발생하는 이벤트
16.	dblclick: 요소를 더블 클릭할 때 발생하는 이벤트
17.	drag: 요소를 드래그할 때 발생하는 이벤트
18.	dragend: 드래그 작업이 끝났을 때 발생하는 이벤트
19.	dragenter: 드래그된 요소가 들어갈 때 발생하는 이벤트
20.	dragexit: 드래그된 요소가 떠났을 때 발생하는 이벤트
21.	dragleave: 드래그된 요소가 떠날 때 발생하는 이벤트
22.	dragover: 드래그된 요소 위로 마우스가 움직일 때 발생하는 이벤트
23.	dragstart: 드래그 작업이 시작될 때 발생하는 이벤트
24.	drop: 드롭 작업이 완료될 때 발생하는 이벤트
25.	durationchange: 미디어의 재생 시간이 변경될 때 발생하는 이벤트
26.	emptied: 미디어의 데이터가 없어졌을 때 발생하는 이벤트
27.	ended: 미디어의 재생이 끝났을 때 발생하는 이벤트
28.	error: 리소스 로드 중에 에러가 발생했을 때 발생하는 이벤트
29.	focus: 요소가 포커스를 받을 때 발생하는 이벤트
30.	focusin: 요소가 포커스를 받을 때 발생하는 이벤트 (버블링)
31.	focusout: 요소가 포커스를 잃을 때 발생하는 이벤트 (버블링)
32.	fullscreenchange: 전체화면 모드 변경 시 발생하는 이벤트
33.	fullscreenerror: 전체화면 모드로 진입하지 못했을 때 발생하는 이벤트
34.	hashchange: URL 해시 부분이 변경될 때 발생하는 이벤트
35.	input: 입력 요소의 값이 변경되었을 때 발생하는 이벤트 (실시간 업데이트)
36.	invalid: 입력 요소의 값이 유효하지 않을 때 발생하는 이벤트
37.	keydown: 키보드의 키를 누를 때 발생하는 이벤트
38.	keypress: 키가 눌리고 놓일 때 발생하는 이벤트 (특정 키는 제외)
39.	keyup: 키보드의 키를 누른 후 놓을 때 발생하는 이벤트
40.	load: 리소스(이미지, 스크립트 등)가 로드되었을 때 발생하는 이벤트
41.	loadeddata: 미디어의 데이터 로딩이 완료될 때 발생하는 이벤트
42.	loadedmetadata: 미디어의 메타데이터 로딩이 완료될 때 발생하는 이벤트
43.	loadstart: 리소스의 로딩이 시작될 때 발생하는 이벤트
44.	message: 메시지가 수신되었을 때 발생하는 이벤트
45.	mousedown: 마우스 버튼을 누를 때 발생하는 이벤트
46.	mouseenter: 마우스 포인터가 요소 안으로 들어갈 때 발생하는 이벤트
47.	mouseleave: 마우스 포인터가 요소 밖으로 나갈 때 발생하는 이벤트
48.	mousemove: 마우스 포인터가 요소 위를 움직일 때 발생하는 이벤트
49.	mouseout: 마우스 포인터가 요소를 떠날 때 발생하는 이벤트
50.	mouseover: 마우스 포인터가 요소 위에 올라갈 때 발생하는 이벤트
51.	mouseup: 마우스 버튼을 눌렀다가 놓을 때 발생하는 이벤트
52.	offline: 오프라인 상태가 될 때 발생하는 이벤트
53.	online: 온라인 상태가 될 때 발생하는 이벤트
54.	pagehide: 페이지가 닫힐 때 발생하는 이벤트
55.	pageshow: 페이지가 나타날 때 발생하는 이벤트
56.	paste: 붙여넣기 전에 발생하는 이벤트
57.	pause: 미디어의 재생이 일시 중지될 때 발생하는 이벤트
58.	play: 미디어의 재생이 시작될 때 발생하는 이벤트
59.	playing: 미디어의 재생이 시작된 후 발생하는 이벤트
60.	popstate: 브라우저의 세션 히스토리가 변경될 때 발생하는 이벤트
61.	progress: 리소스 로딩 진행 상황이 업데이트될 때 발생하는 이벤트
62.	ratechange: 미디어의 재생 속도가 변경될 때 발생하는 이벤트
63.	resize: 브라우저 창의 크기가 변경될 때 발생하는 이벤트
64.	reset: 폼을 초기화할 때 발생하는 이벤트
65.	scroll: 요소 내부의 내용이 스크롤될 때 발생하는 이벤트
66.	seeked: 미디어의 탐색이 완료될 때 발생하는 이벤트
67.	seeking: 미디어의 탐색이 시작될 때 발생하는 이벤트
68.	select: 입력 요소 내부 텍스트가 선택될 때 발생하는 이벤트
69.	show: 숨겨진 요소가 표시될 때 발생하는 이벤트
70.	stalled: 미디어의 로딩이 지연될 때 발생하는 이벤트
71.	storage: 웹 스토리지 데이터가 변경될 때 발생하는 이벤트
72.	submit: 폼을 제출할 때 발생하는 이벤트
73.	suspend: 미디어의 로딩이 일시 중지될 때 발생하는 이벤트
74.	timeupdate: 미디어의 재생 시간이 업데이트될 때 발생하는 이벤트
75.	toggle: 요소의 토글 상태가 변경될 때 발생하는 이벤트
76.	touchcancel: 터치 디바이스에서 터치가 취소될 때 발생하는 이벤트
77.	touchend: 터치 디바이스에서 터치가 종료될 때 발생하는 이벤트
78.	touchmove: 터치 디바이스에서 터치 동작이 진행될 때 발생하는 이벤트
79.	touchstart: 터치 디바이스에서 터치가 시작될 때 발생하는 이벤트
80.	transitionend: CSS 트랜지션 종료 시 발생하는 이벤트
81.	unload: 페이지를 떠날 때 발생하는 이벤트
82.	volumechange: 미디어의 볼륨이 변경될 때 발생하는 이벤트
83.	waiting: 미디어의 재생이 일시 중지될 때 발생하는 이벤트
84.	wheel: 마우스 휠을 돌릴 때 발생하는 이벤트
