1. basic_ - 언어 / 프레임워크별 기본적으로 사용하는 문법 / 메소드
2. useful_ - 언어 / 프레임워크 별 유용한 문법 모음
3. useful_function - 언어 / 프레임워크 별 유용한 메소드 모음
